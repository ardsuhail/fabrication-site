# Metal Kohinoor - Fabrication Services Website

[![Next.js](https://img.shields.io/badge/Next.js-16.2.1-black?logo=next.js)](https://nextjs.org/)
[![React](https://img.shields.io/badge/React-19.0.0-blue?logo=react)](https://reactjs.org/)

A modern, SEO-optimized website for Metal Kohinoor - a professional fabrication service provider specializing in ACP cladding, glass partitions, aluminium windows, and UPVC work in Delhi NCR.

## 🌟 Features

- **🚀 Next.js 16** - Server-side rendering and static site generation
- **🎨 Tailwind CSS** - Utility-first styling with custom theme
- **📱 Fully Responsive** - Mobile-first design that works on all devices
- **🔍 SEO Optimized** - Complete metadata, JSON-LD schemas, robots.txt, sitemap.xml
- **💬 WhatsApp Integration** - Direct chat for quotes and inquiries
- **📸 Media Support** - Images and videos with gallery view
- **⚡ Performance Optimized** - Fast loading with Next.js Image optimization
- **📍 Local Business SEO** - Google Maps, business hours, location data
- **📊 Structured Data** - Rich snippets for better search results

## 📋 Pages

| Page | Description |
|------|-------------|
| **Home** | Hero section, services overview, stats, testimonials |
| **About** | Company history, owner profile, achievements |
| **Services** | Detailed service pages with FAQs and benefits |
| **Projects** | Portfolio with filtering by category and location |
| **Project Detail** | Individual project pages with images/videos |
| **Contact** | Contact form, WhatsApp, call, and location map |

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: JavaScript/React
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Fonts**: Geist (Google Fonts)
- **Deployment**: Vercel / Any hosting platform
